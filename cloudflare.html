Using Cloudflare for DNS & Security

Secure and Speed Up Your Site

  1. Register a domain (e.g., via Namecheap or Google Domains).
  2. Create an account at cloudflare.com.
  3. Add your domain to Cloudflare and let it scan for DNS records.
  4. Update your domainâ€™s nameservers to the ones Cloudflare gives you.
  5. Under DNS settings, add a CNAME record pointing your domain to your GitHub Pages URL.
  6. Enable "Always Use HTTPS" and set SSL mode to "Full".

Now your site is not only free, but also faster and more secure!
