Hosting with GitHub Pages

Step-by-Step Instructions

  1. Create a GitHub account at github.com.
  2. Click "New repository", name it like yourusername.github.io.
  3. Upload your HTML files (like `index.html`) to the repository.
  4. Click "Settings" → "Pages" → Choose the branch as source (usually main).
  5. Click Save. Your site will be available at https://yourusername.github.io

Now your site is live! GitHub Pages provides secure, fast, and free hosting.
